O que é Css?

Css é uma linguagem de estilo feita para controlar a forma como certos elementos do HTML de uma página web são visualizados no front-end. Você pode influenciar a aparência de listas, elementos de formulário e tabelas. Essa linguagem é uma das três importantes para web, ela define a aparência de uma página web.

O Princípio

Você pode definir como certos elementos HTML serão exibidos usando Css. Essas regras consistem em duas partes. O seletor Css permite especificar quais elementos serão submetidos as regras do Css. Você pode usar chaves para especificar como os elementos serão exibidos. As declarações consistem em uma propriedade CSS e um valor.

Maneiras de incluir Css em HTML

  1. Css Externo
  2. Css Interno
  3. Css Inline

Css Externo

No Css externo, as instruções Css ficam em um arquivo separado com extensão ".css" e depois esse arquivo é conectado ao documento HTML.

<link href="style.css" rel="stylesheet">

Vantagens do Css Externo

Vantagem Descrição
Organização O código fica separado do HTML.
Reutilização O mesmo arquivo Css pode ser usado em várias páginas.
Facilidade Mais fácil de editar e manter a consistência visual.

Css Interno

No Css interno, as regras Css são definidas dentro da própria página HTML. Geralmente elas ficam dentro da tag style no head da página.

<style>
body{
background-color: lightblue;
}
</style>

Css Inline

No Css inline, o estilo é colocado diretamente em um elemento HTML. Esse jeito não é muito recomendado porque a reutilização das regras CSS é perdida.

<h1 style="color: blue;">Título</h1>

Seletores Css

Os seletores servem para escolher quais elementos HTML receberão as regras Css.

Seletor Exemplo Descrição
Tipo h1 Seleciona todos os elementos de acordo com a tag.
Classe .titulo Seleciona elementos com base nas classes CSS.
ID #menu Seleciona elemento com o id correspondente.
Universal * Seleciona todos os elementos.

Seletores Avançados e Combinações

Além dos seletores básicos, o CSS permite seleções muito mais complexas combinando regras.

Seletor Exemplo Descrição
Atributo [type="text"] Seleciona elementos com base no valor de um de seus atributos.
Irmão Adjacente h1 + p Seleciona elementos que seguem imediatamente outro elemento.
Irmão Geral h1 ~ p Seleciona elementos que seguem outro elemento em algum momento.
Filho Direto body > ul Seleciona elementos que são filhos diretos.
Descendente body ul Seleciona elementos que são descendentes (mesmo que não diretos).

Pseudo-elementos

Enquanto as pseudo-classes selecionam elementos em estados, os pseudo-elementos abordam partes específicas de um elemento selecionado e podem até criar novos elementos não presentes no HTML.

Pseudo-elemento Função
::before Cria um elemento antes do elemento selecionado.
::after Cria um elemento depois do elemento selecionado.
::first-letter Refere-se à primeira letra do texto.
::first-line Refere-se à primeira linha de texto.
::selection Refere-se à parte do texto selecionada pelo usuário.

Cascata e Especificidade

O Css funciona em forma de cascata. Devido a isso, várias regras podem se aplicar a um único elemento. Se houver regras conflitantes, a regra mais específica tem prioridade.

Cálculo de Especificidade

O cálculo da especificidade é baseado em três contadores (A, B e C), iniciados em 0:

  • Contador A: Incrementado em 1 para cada seletor de ID.
  • Contador B: Incrementado em 1 para cada seletor de classe, atributo ou pseudo-classe.
  • Contador C: Incrementado em 1 para cada seletor de tipo ou pseudo-elemento.

Exemplo: O seletor #chapter p resulta em A=1, B=0, C=1 (Especificidade: 101). Seletores universais não contam no cálculo.

Herança

Alguns valores definidos via regras CSS em elementos pais são herdados por seus elementos filhos. Por exemplo, a cor e a fonte passam adiante. Propriedades como bordas (border) não são herdadas.

Cores no Css

O Css permite definir cores de várias formas diferentes:

  • Nomes de cores: Mais de 140 nomes predefinidos (ex: yellow, blue).
  • Valores RGB: Composição de vermelho, verde e azul (0 a 255).
  • Valores Hexadecimais: Valor de 6 dígitos (ex: #FFFF00).
  • RGBA: Valor RGB com um valor alfa para opacidade.
  • HSL e HSLA: Baseado em matiz, saturação e luminosidade (e opacidade no HSLA).

Formatação de Texto

Você pode influenciar a aparência básica do texto usando propriedades CSS:

  • font-family: Define a família da fonte.
  • font-size: Define o tamanho da fonte.
  • font-style: Estilo (ex: itálico).
  • font-weight: Peso/negrito.
  • text-align: Alinhamento horizontal.

Links no Css

Os links podem ser personalizados através de pseudo-classes.

Pseudo-classe Descrição
:link Permite formatar links que ainda não foram visitados.
:visited Permite formatar links que já foram visitados.
:hover Permite formatar links quando "tocados" com o mouse.
:active Permite formatar links no momento em que são clicados.
:focus Permite formatar links quando eles recebem o foco.

Listas no Css

O Css permite personalizar listas ordenadas e não ordenadas, principalmente os marcadores.

Imagens e Posicionamento de Marcadores

  • list-style-image: Passa o caminho ou URL para a imagem desejada usada como marcador.
  • list-style-position: Personaliza a posição. O valor outside deixa o marcador à esquerda (padrão). O valor inside garante que o marcador fique recuado dentro do bloco de texto.

Lista não ordenada

  • HTML
  • CSS
  • JavaScript

Lista ordenada

  1. HTML
  2. CSS
  3. JavaScript

Tabelas no Css

As tabelas sempre parecem simples demais sem CSS, mas podem ser facilmente personalizadas.

Propriedade Função
border Usada para projetar bordas (largura, estilo e cor).
padding Define quanto espaço deve existir entre o conteúdo de uma célula e sua borda.
border-collapse Usada para "colapsar" as bordas para que as células "compartilhem" uma borda.

Sistemas de Layout (Seção 3.4)

Usando CSS, você pode posicionar com precisão os elementos HTML em uma página da web e influenciar o layout.

Elementos Block-Level vs. Inline-Level

Basicamente, o HTML distingue entre elementos de nível de bloco e elementos de nível em linha:

  • Block-level: São sempre exibidos em uma nova linha e organizados verticalmente.
  • Inline-level: São exibidos horizontalmente onde aparecem no fluxo de texto.

Com a propriedade CSS display, você pode personalizar isso: os valores block ou inline forçam o comportamento correspondente.

Tipos de Posicionamento

Você tem maneiras diferentes de posicionar elementos usando a propriedade position:

Valor Descrição
static Posicionados conforme aparecem no código HTML (no fluxo normal).
relative Posicionados de forma relativa (para cima, direita, baixo, esquerda) baseados em sua posição normal.
absolute Retirados do fluxo normal e posicionados em relação ao elemento pai.
fixed Posicionados em relação à janela do navegador (viewport).
sticky Comportam-se como fixed, mas rolam apenas até um ponto e então permanecem fixos.

Float Layout

A propriedade float pode ser usada para definir elementos para a borda esquerda (left) ou direita (right) do bloco circundante. Por muito tempo, foi a arma de escolha para organizar elementos, mas hoje temos alternativas melhores.

Flexbox Layout

Introduzido com o CSS3, o layout flexbox permite que um contêiner ajuste dinamicamente a largura e a altura de seus itens para preencher o espaço disponível. Ao contrário dos layouts regulares, o flexbox é independente de direção (arranjos verticais e horizontais).

Grid Layout

Para layouts complexos, o grid layout (grade) é o mais adequado. Em contraste com o flexbox (unidimensional), este é bidimensional, envolvendo o posicionamento de itens em uma grade dentro de um contêiner, facilitando a implementação de designs avançados.

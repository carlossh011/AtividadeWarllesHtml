O que é CSS?

CSS é uma linguagem de estilo que controla a apresentação e a aparência dos elementos HTML no front-end de uma página web.

O Princípio

As regras CSS possuem duas partes: o seletor (indica quais elementos receberão o estilo) e as declarações entre chaves (compostas por uma propriedade e um valor correspondente).

Inclusão de CSS no HTML

  1. CSS Externo: Arquivo .css separado, conectado via tag <link>.
  2. CSS Interno: Regras definidas na tag <style>, dentro do head.
  3. CSS Inline: Estilo aplicado diretamente no elemento HTML. Pouco recomendado por dificultar a reutilização.
  4. Regra @import: Usada dentro de arquivos CSS ou da tag <style> para importar outros arquivos de estilo.

Vantagens do CSS Externo

Vantagem Descrição
Organização Separa o código de estilo do HTML.
Reutilização Um único arquivo serve para várias páginas.
Manutenção Facilita a edição e garante consistência visual.

Seletores CSS

Seletor Exemplo Descrição
Tipo h1 Seleciona pela tag HTML.
Classe .titulo Seleciona por classes.
ID #menu Seleciona pelo identificador único.
Universal * Seleciona todos os elementos.

Seletores Avançados

Seletor Exemplo Descrição
Atributo [type="text"] Seleciona pelo valor de um atributo.
Irmão Adjacente h1 + p Seleciona o elemento imediatamente seguinte.
Irmão Geral h1 ~ p Seleciona irmãos que vêm depois em qualquer momento.
Filho Direto body > ul Seleciona apenas os filhos diretos.
Descendente body ul Seleciona qualquer descendente.

Pseudo-elementos

Permitem estilizar partes específicas de um elemento ou criar novos elementos virtuais.

Pseudo-elemento Função
::before / ::after Insere conteúdo antes ou depois do elemento.
::first-letter / ::first-line Estiliza a primeira letra ou linha.
::selection Estiliza o texto selecionado pelo usuário.

Cascata e Especificidade

O CSS aplica regras em cascata. Em caso de conflito, a regra mais específica vence.

  • Contador A: Seletores de ID.
  • Contador B: Classes, atributos ou pseudo-classes.
  • Contador C: Tipos ou pseudo-elementos.

Herança

Algumas propriedades (como cor e fonte) são herdadas de elementos pais para filhos, enquanto outras (como bordas) não são.

Cores e Texto

  • Cores: Podem ser definidas por nomes, RGB, Hexadecimal, RGBA, HSL ou HSLA.
  • Fonte: Controlada por font-family, font-size, font-style (itálico) e font-weight (negrito).
  • Formatação: text-transform (maiúsculas/minúsculas), text-decoration (sublinhado), line-height (espaço entre linhas), letter-spacing e text-align (alinhamento).

Links

Pseudo-classe Descrição
:link / :visited Estiliza links não visitados e já visitados.
:hover / :active Estiliza quando o mouse passa por cima e no momento do clique.
:focus Estiliza quando o link recebe foco.

Listas e Tabelas

  • Listas: Personalizadas com list-style-image (imagem como marcador) e list-style-position (marcador dentro ou fora do bloco).
  • Tabelas: Estilizadas com border (bordas), padding (espaçamento interno) e border-collapse (une bordas vizinhas).

Sistemas de Layout

Controlam o posicionamento e a organização dos elementos na página.

Block-Level vs Inline-Level

  • Block-level: Ocupam a linha inteira verticalmente (ex: divs, parágrafos).
  • Inline-level: Ocupam apenas o espaço do seu conteúdo horizontalmente (ex: spans, links).

Posicionamento (position)

Valor Descrição
static Fluxo normal padrão do HTML.
relative Posição relativa ao seu local original.
absolute Posição relativa ao elemento pai posicionado mais próximo.
fixed Fixo em relação à tela do navegador.
sticky Rola normalmente até um ponto específico e então fica fixo.

Layouts em Formulários e Elementos

  • Float Layout: Empurra elementos para a borda esquerda ou direita. Usado historicamente para colocar labels e inputs lado a lado.
  • Flexbox Layout: Layout unidimensional. Ajusta dinamicamente a largura e altura dos itens (flex items) dentro de um contêiner (flex container) para preencher o espaço, podendo ser alinhado em linha ou coluna.
  • Grid Layout: Layout bidimensional. Permite organizar elementos complexos através da definição de linhas e colunas dentro de uma grade.

Perspectivas Futuras do CSS (Outlook)

  • Animações: Transições suaves entre propriedades e efeitos visuais.
  • Design Responsivo: Uso de Media Queries para adaptar o layout a diferentes tamanhos de tela (smartphones, tablets, monitores).
  • Frameworks: Ferramentas como Bootstrap e Materialize que oferecem estilos prontos para acelerar o desenvolvimento.

O que é Css?

Css é uma linguagem de estilo feita para controlar a forma como certos elementos do HTML de uma página web são visualizados no front-end. Você pode influenciar a aparência de listas, elementos de formulário e tabelas. Essa linguagem é uma das três importantes para web, ela define a aparência de uma página web.

O Princípio

Você pode definir como certos elementos HTML serão exibidos usando Css. Essas regras consistem em duas partes. O seletor Css permite especificar quais elementos serão submetidos as regras do Css. Você pode usar chaves para especificar como os elementos serão exibidos. As declarações consistem em uma propriedade CSS e um valor.

Maneiras de incluir Css em HTML

  1. Css Externo
  2. Css Interno
  3. Css Inline

Css Externo

No Css externo, as instruções Css ficam em um arquivo separado com extensão ".css". Depois esse arquivo é conectado ao documento HTML usando a tag link.

<link href="style.css" rel="stylesheet">

Vantagens do Css Externo

Vantagem Descrição
Organização O código fica separado do HTML
Reutilização O mesmo arquivo Css pode ser usado em várias páginas
Facilidade Mais fácil de editar e manter

Css Interno

No Css interno, as regras Css são definidas dentro da própria página HTML. Geralmente elas ficam dentro da tag style no head da página.

<style>
body{
background-color: lightblue;
}
</style>

Css Inline

No Css inline, o estilo é colocado diretamente no elemento HTML. Esse jeito não é muito recomendado porque dificulta a reutilização do código.

<h1 style="color: blue;">Título</h1>

Seletores Css

Os seletores servem para escolher quais elementos HTML receberão as regras Css.

Seletor Exemplo Descrição
Tipo h1 Seleciona todos os elementos h1
Classe .titulo Seleciona elementos com classe titulo
ID #menu Seleciona elemento com id menu
Universal * Seleciona todos os elementos

Exemplos de Seletores

Código Resultado
h1 Seleciona todos os títulos h1
p Seleciona todos os parágrafos
.caixa Seleciona elementos da classe caixa
#menu Seleciona elemento com id menu

Cascata e Especificidade

O Css funciona em forma de cascata. Isso significa que várias regras podem ser aplicadas ao mesmo elemento. A regra mais específica geralmente terá prioridade.

h1{
color: blue;
}

h1{
color: green;
}

Nesse exemplo, a cor verde será aplicada porque a última regra foi a mais recente.

Herança

Algumas propriedades Css podem ser herdadas pelos elementos filhos. Por exemplo, cor e fonte podem passar do elemento pai para os elementos filhos.

Cores no Css

O Css permite definir cores de várias formas diferentes.

  • Nomes de cores
  • Valores RGB
  • Valores Hexadecimais
  • RGBA
  • HSL
Tipo Exemplo
Nome blue
RGB rgb(255,0,0)
Hexadecimal #FF0000

Tabela de Cores

Cor Código
Azul blue
Vermelho #FF0000
Verde rgb(0,255,0)
Amarelo yellow

Formatação de Texto

O Css também permite modificar fontes, tamanho do texto, alinhamento e espaçamento.

  • font-family
  • font-size
  • font-style
  • font-weight
  • text-align
p{
font-family: Arial;
font-size: 16px;
text-align: justify;
}

Links no Css

Os links podem mudar de aparência dependendo da interação do usuário.

Pseudo-classe Função
:link Link normal
:visited Link visitado
:hover Quando o mouse passa
:active Quando o link é clicado

Estados do Link

Estado Descrição
:link Link normal
:visited Link já visitado
:hover Mouse em cima do link
:active Momento do clique

Listas no Css

O Css permite personalizar listas ordenadas e não ordenadas.

Lista não ordenada

  • HTML
  • CSS
  • JavaScript

Lista ordenada

  1. HTML
  2. CSS
  3. JavaScript

Tabelas no Css

As tabelas podem ficar mais bonitas usando Css. É possível modificar bordas, cores, alinhamento e espaçamento.

Linguagem Função
HTML Estrutura
CSS Estilo
JavaScript Interatividade

Propriedades usadas em Tabelas

Propriedade Função

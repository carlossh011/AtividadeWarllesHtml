Resumo

O HTML é uma das linguagens principais usadas para criar sites. Ele serve para organizar a estrutura da página. Surgiu em 1992 e hoje usamos o HTML5, que trouxe várias melhorias.

Elementos e Atributos

Os elementos são definidos por tags, como <h1> e </h1>. Normalmente possuem uma tag de abertura e outra de fechamento.

Os atributos ficam dentro da tag de abertura e servem para adicionar informações extras, como o atributo href nos links.

Principais Elementos

Títulos: h1 até h6

Parágrafos: p

Formatação de texto: b e i

Listas: ul e ol

Links: a

Imagens: img

Tabelas: table

Formulários: form

Como Salvar

Um arquivo HTML é salvo com a extensão .html. Depois de salvar, basta abrir no navegador.

Organizar cada projeto em uma pasta separada.

Outros Pontos

Existem elementos de bloco e inline.

div: usado para agrupar conteúdo em bloco

span: usado para agrupar conteúdo na mesma linha

O HTML5 também trouxe novos elementos, como:

header

footer

nav

section
